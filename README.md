# yumWorld React App

## What Is This Project About
This website is a food app that uses React and API calls, to render meal instructions to the user based on the selected country and meal.

## Technologies used
* HTML
* CSS / Flexbox
* Vanilla JavaScript
* React
* React Router
* NPM
